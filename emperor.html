How the Emperor Share System Works
1
Your Main Code — permanent login. Format: name-xxxxx. Keep it safe — never share it.
2
Emperor Code — the code issued when you earned your final medal. This logs you into this portal only. Never share it with anyone.
3
Tier Codes — each tier (STR, BRZ, SLV, GLD, ELT) has its own share code. Give that code to friends to confirm. When they do, your tier count rises.
4
Share Messages — tap any tier's share button to send a pre-written message with your code via WhatsApp, SMS, Telegram, and more.
5
Rewards Auto-Unlock — when a tier fills up, Puzzlr detects it and unlocks your reward automatically next time you go online in the game.
